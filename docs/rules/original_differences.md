# Original Differences
A list with the main differences between the original game and this remake.

### Block tiles
-  Original: 
    - Block tiles block their position and their neighbour positions (up, down, left right). 
- Remake:
    - Block tiles only block their position.

### Reply Abilities
- Original:
    - Reply abilities cannot be activated when doing a GUARD command.
- Remake:
    - Reply abilities can be activated both when doing a WAIT command and a GUARD command[^1].

### Time Wizard Ability
- Original:
    - Time Wizard ability destroys the opponent monster with the lowest attack.
- Remake:
    - Time Wizard ability (DIMKILLWEAKEST) destroys the monster with the lowest attack (between player and opponent monsters)[^2].

### Dark-eyes Illusionist Ability
- Original:
    - Dark-eyes Illusionist ability reduces the damage to 0 when attacked.
- Remake:
    - Dark-eyes Illusionist ability allows it to select a monster and negate that monster attacks and abilities.[^3]

### Knight of Twin Swords Ability
- Original
    - Knight of Twin Swords ability allows to increase the attack damage in 30 or less.
- Remake
    - Knight of Twin Swords ability allows to perform an attack up to 3 times per turn.[^4]

[^1]: Some reply abilities are borderline useless if they can only be activated during WAIT commands, like REDUCEDAMAGE and ADDFOEDEFENSE.

[^2]: The original Time Wizard ability is too broken. A dicepool filled with Time Wizards can effectively invalidate any opponent dicepool.

[^3]: The original ability of Dark-eyes Illusionist is too weak compared to its cost, specially compared to Strike Ninja that has the same ability for a lower cost.

[^4]: The remake ability is more in line with the ability description, and the anime ability.
